# 算法与数据结构

本项目的主要目的是学习基本的算法与数据结构，并通过 Python 代码实现。

目前已经收录了如下的算法和数据结构：

- **算法**
  - 排序算法
  - 搜索算法
- **数据结构**
  - 链表

所有已实现的代码都已通过单元测试 (所有的测试文件在目录 tests 下)，你可以把项目 clone 到本地，然后在终端输入命令 `python -m unittest <test_file_path>` 进行测试。如果你要测试所有的排序算法，可以使用如下命令：

```bash
python -m unittest tests/test_sort.py
```
